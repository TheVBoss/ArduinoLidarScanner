## ArduinoLidarScanner
An Arduino-based lidar scanner using servos and a sensor!

# HOW TO USE
1. Download Arduino IDE and Processing software.
2. Plug the Arduino Uno into your computer.
3. First, run the Arduino Sketch in Arduino IDE (ensure Serial Monitor is not displayed).
4. Then run the Processing Sketch in Processing.
